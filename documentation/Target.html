clickTimes(value)

setter

Sets Target.clickTimes to a first in, first out queue of Target.ClickTime values with a maximum length of 2, or clears the click time queue. When the maximum length of click times is set, a Target.ClickInterval is created and added to Target.clickIntervals, then the first click time in the queue is removed. If an empty array is provided for value, the click times queue will be cleared. Updates Timer.domElements.averageClick.element.textContent.

Name Type Description

A Target.ClickTime or an empty array to clear the click times queue.
