Class: Items

Items(timer, logger, score)

Manages items and interacts with DOM elements related to them. Item ClickEventListener delegatees can accept or ignore browser click events depending on the item state. Item DOM elements are dynamically added to the DOM. Item event delegatees are dynamically added to the class object when the item elements are added to the DOM.

itemsElementClick()

A ClickEventListener delegatee that determines if the item is accepting clicks to conditionally call Score.addPointsToSessionScore to handle the points value, then creates a click log entry. The points value of a clicked item is equal to item.spawnIntervalSeconds. One point is deducted per second that a click time follows item.spawnIntervalSeconds. A maximum value of item.spawnTimeSeconds can be deducted from the points value, meaning that the minimum number of points that can be scored is 0.

The time, relative to the value returned by Timer.seconds, at which the event delegatee will begin accepting click events when a new session is started. The interval for which the item's event delegatee ignores click events after it has accepted a click event. This is also the points value provided to Score.addPointsToSessionScore when the event delegatee accepts a click.
