Arena FPS Item Timing Trainer

By Brandan Glendenning (GitHub)

When a session is started, each item spawns immediately. Picking up an item conditionally adds points to the session score. An item's maximum points value is equal to its spawn delay time, as indicated on the item. A point is deducted from the maximum points value for each second elapsed after the item spawns that the item is not picked up. The maximum value that can be deducted is the item's spawn delay time. Points are multiplied by the timescale multiplier value before being added to the session score.

Start begins or continues a paused session. - & + manipulate the rate at which the timer elapses. Pause pauses the timer. End ends the session, sets the session score to 0, and resets the target to its original position. Check Enable randomized target to display an interactive target that is assigned random positions and dimensions after each click. Targets clicked are worth 1 point, multiplied by the timescale multiplier value.
